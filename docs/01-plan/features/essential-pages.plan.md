> **Feature**: essential-pages
> **상위 계획**: AdSense 승인 로드맵 > PDCA-6
> **작성일**: 2026-02-25
> **우선순위**: P1 (필수)

---

## 1. 목적

AdSense 재신청 전 마지막 콘텐츠/정책 보강 단계. 씬 콘텐츠(thin content) 해소, GDPR 쿠키 동의 배너 구현, E-E-A-T 신호 추가, 게임 허브 한국어 소개 보강을 통해 AdSense 승인 요건을 완전히 충족한다.

---

## 2. 현상 분석

### 2.1 블로그 콘텐츠 현황

| 포스트 | 줄 수 | 추정 글자 수 | 상태 |
|--------|-------|-------------|------|
| hello-world.mdx | 29 | ~400자 | **씬 콘텐츠** |
| getting-started.mdx | 45 | ~600자 | **씬 콘텐츠** |
| speak-100-days-review.mdx | 64 | ~2,000자 | 양호 |
| iterm2-korean-fix.mdx | 66 | ~1,800자 | 양호 |
| bullterrier-houhou-review.mdx | 61 | ~1,500자 | 양호 |
| joker-gi-review.mdx | 80 | ~2,500자 | 양호 |
| nextjs-mdx-blog-guide.mdx | 129 | ~3,500자 | 좋음 |

- 총 7개 포스트 중 **2개가 씬 콘텐츠** (1,000자 미만)
- AdSense 권장: 포스트당 1,500자+, 총 10-15개 이상

### 2.2 쿠키 동의 배너

- **현재 상태**: 미구현
- **필요 이유**: Google AdSense 사용 시 GDPR/ePrivacy 준수 필수
- 개인정보처리방침(privacy)에 쿠키 사용을 명시하고 있으나, 사용자 동의 수집 메커니즘 없음

### 2.3 게임 허브 페이지

- **현재 소개 텍스트**: "Simple web games to enjoy." (영어, 1줄)
- **메타 description**: "A collection of simple web games" (영어)
- 한국어 사이트인데 게임 허브가 영어로만 되어 있음
- 소개 콘텐츠 부족 (씬 콘텐츠 판정 가능)

### 2.4 E-E-A-T 신호

| 항목 | 현재 상태 |
|------|----------|
| 저자 프로필 (블로그) | 없음 |
| 작성일/수정일 표시 | frontmatter에 date 있으나 updatedAt 없음 |
| 출처 인용 | 일부 포스트만 |
| About 페이지 저자 소개 | 사이트 소개만 있고 개인 프로필 부족 |

## 4. 범위 (FR)

### FR-01: 기존 블로그 씬 콘텐츠 보강

| 포스트 | 현재 | 보강 방향 | 목표 |
|--------|------|----------|------|
| hello-world.mdx | ~400자 | 사이트 비전, 프로젝트 배경, 개발 철학, 향후 로드맵 확장 | 1,500자+ |
| getting-started.mdx | ~600자 | 개발 환경 상세 설명, 폴더 구조 해설, 기여 가이드 추가 | 1,500자+ |

### FR-02: 쿠키 동의 배너 구현

| 작업 | 상세 | 완료 기준 |
|------|------|----------|
| CookieConsent 컴포넌트 | 하단 고정 배너, 수락/거부 버튼 | 첫 방문 시 표시 |
| localStorage 동의 저장 | `needcash-cookie-consent` 키 | 동의 후 미표시 |
| AdSense 스크립트 조건부 로드 | 동의 전 AdSense 스크립트 미로드 | 동의 후에만 로드 |
| 개인정보처리방침 링크 | 배너에서 /privacy 링크 | 클릭 시 이동 |

### FR-03: 게임 허브 한국어 소개 보강

| 작업 | 상세 | 완료 기준 |
|------|------|----------|
| 메타데이터 한국어 전환 | title, description 한국어로 | metadata에 반영 |
| 소개 텍스트 보강 | 게임 카테고리 설명, 사이트 맥락 소개 | 300자+ 한국어 텍스트 |
| 카테고리별 소개 | 두뇌 훈련, 확률/랜덤, 기타 카테고리 설명 | 각 카테고리 100자+ |

### FR-04: 신규 블로그 포스트 3개 작성

게임과 연결된 교육적 콘텐츠로 총 포스트 수를 10개로 확보.

| # | 제목 (안) | 연결 게임 | 목표 |
|---|----------|----------|------|
| 1 | "반응 속도의 과학: 뇌는 얼마나 빠를까?" | Reaction Test | 2,000자+ |
| 2 | "색감 테스트로 알아보는 색각의 세계" | Color Sense | 2,000자+ |
| 3 | "기억력과 패턴 인식: 사이먼 게임의 인지과학" | Color Memory | 2,000자+ |

각 포스트는 과학적 배경 + 게임 링크 + 실생활 활용으로 구성.

### FR-05: E-E-A-T 신호 추가

| 작업 | 상세 | 완료 기준 |
|------|------|----------|
| 저자 프로필 컴포넌트 | 블로그 포스트 하단에 저자 이름, 한 줄 소개 | 모든 포스트에 표시 |
| updatedAt 지원 | MDX frontmatter에 updatedAt 필드 추가, UI에 표시 | 수정일 있는 글에 표시 |
| About 페이지 개인 소개 | 운영자 경력/전문성 소개 문단 추가 | 200자+ 개인 소개 |
