## 안녕하세요, NeedCash입니다

NeedCash 블로그에 오신 것을 환영합니다. 이 글에서는 NeedCash가 어떤 프로젝트인지, 왜 만들게 되었는지, 그리고 앞으로 어떤 방향으로 발전시켜 나갈 것인지를 소개하겠습니다.

## NeedCash를 시작한 이유

개발자로서 다양한 사이드 프로젝트를 진행하다 보면, 각각의 프로젝트가 흩어져 관리하기 어려워지는 문제를 자주 경험하게 됩니다. 블로그는 여기, 포트폴리오는 저기, 실험적인 프로젝트는 또 다른 곳에... 이런 상황을 해결하고 싶어서 NeedCash를 만들었습니다.

NeedCash는 **프로토타입 허브**입니다. 블로그, 웹 게임, 이력서, 광고 랜딩 페이지 등 다양한 프로토타입을 하나의 공간에서 관리합니다. "NeedCash"라는 이름은 개발자라면 누구나 공감할 수 있는 현실적인 동기에서 가져왔습니다.

## 어떤 콘텐츠를 다루나요?

### 블로그

개발 과정에서 배운 것들을 기록합니다. 주요 주제는 다음과 같습니다:

- **개발 팁**: Next.js, TypeScript, Tailwind CSS 등 웹 개발 기술에 대한 실전 가이드
- **제품 리뷰**: 주짓수 도복, 일본어 학습 앱 등 직접 사용해본 제품의 솔직한 후기
- **라이프스타일**: 개발자의 일상과 취미 생활에 대한 이야기

### 게임

단순한 오락을 넘어 인지 능력 훈련을 목적으로 한 웹 게임을 제공합니다:

- **반응속도 테스트**: 시각 자극에 대한 반응 시간을 측정합니다
- **색감 테스트**: 미세한 색상 차이를 구분하는 능력을 테스트합니다
- **색상 기억력 게임**: 사이먼(Simon) 스타일의 패턴 기억 게임입니다
- **주사위 굴리기, 로또 번호 생성기**: 확률과 랜덤의 세계를 체험합니다

## 기술 스택

이 사이트는 최신 웹 기술로 구축되었습니다:

- **Next.js 15** (App Router) - React 기반의 풀스택 프레임워크
- **TypeScript 5** - 타입 안전성을 보장하는 JavaScript 슈퍼셋
- **Tailwind CSS 4** - 유틸리티 퍼스트 CSS 프레임워크
- **MDX** - 마크다운 안에서 React 컴포넌트를 사용할 수 있는 포맷
- **Framer Motion** - 부드러운 스크롤 인터랙션과 애니메이션

이 조합을 선택한 이유는 빠른 개발 속도와 뛰어난 성능을 동시에 달성할 수 있기 때문입니다. Next.js의 정적 내보내기(Static Export) 기능을 활용하여 Cloudflare Pages에서 빠르게 서빙하고 있습니다.

## 디자인 시스템

NeedCash는 하나의 고정된 디자인이 아닌, **4가지 디자인 x 4가지 테마 = 16가지 조합**의 멀티 디자인 시스템을 갖추고 있습니다. Editorial, Bento, Brutalist, Glass 네 가지 디자인 스타일 중 취향에 맞는 것을 선택하고, 각 디자인에 맞는 4가지 테마 중 하나를 골라 자신만의 경험을 만들 수 있습니다.
