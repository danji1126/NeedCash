## 로컬 개발 환경

프로젝트를 로컬에서 실행하려면 다음 단계를 따르세요.

### 1. 저장소 클론

```bash
git clone https://github.com/needcash/needcash.git
cd needcash/apps/web
```

### 2. 의존성 설치

```bash
pnpm install
```

### 3. 개발 서버 시작

```bash
pnpm dev
```

브라우저에서 `http://localhost:3000`으로 접속하면 사이트를 확인할 수 있습니다.

## 프로젝트 구조

```
apps/web/
  app/          → 페이지 라우팅
  components/   → 재사용 컴포넌트
  content/      → MDX 블로그 콘텐츠
  lib/          → 유틸리티 함수
```

각 디렉토리의 역할을 이해하고 나면 새로운 페이지나 컴포넌트를 추가하는 것이 쉬워집니다.
